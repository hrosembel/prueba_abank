# Documentación de la API REST

## Introducción

Esta documentación describe una API REST construida con .NET 8, Dapper, y MySQL. La solución sigue una arquitectura de tres capas:

1. **Capa de Presentación**: Controladores y puntos finales de la API (PruebaABANK.API).
2. **Capa de Aplicación**: Lógica de negocio y servicios. (PruebaABANK.BLL)
3. **Capa de Datos**: Acceso a datos usando Dapper. (PruebaABANK.DAL)
